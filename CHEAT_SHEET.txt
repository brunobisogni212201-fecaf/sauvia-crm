  (deve retornar HTML, não erro 404)

  Teste 3: OAuth Metadata
  ──────────────────────
  $ curl http://localhost:3000/.well-known/oauth-protected-resource/mcp | jq
  (deve retornar JSON com scopes)

  Teste 4: MCP com Token
  ─────────────────────
  $ CLERK_TOKEN="seu_token_aqui"
  $ curl -X POST http://localhost:3000/mcp/http \
      -H "Authorization: Bearer $CLERK_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{"jsonrpc":"2.0","id":1,"method":"tools/list"}'

┃ 8. TROUBLESHOOTING RÁPIDO                                              ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  ❌ "ModuleNotFoundError: cryptography"
  → pip install cryptography==42.0.0 --force-reinstall

  ❌ ".well-known retorna 404"
  → cd apps/web && pnpm build && pnpm dev

  ❌ "Cannot find @clerk/mcp-tools"
  → cd apps/web && pnpm install @clerk/mcp-tools

  ❌ "Token inválido"
  → Fazer login novamente em http://localhost:3000
  → Verificar CLERK_JWT_KEY está correto

  ❌ Backend não inicia
  → Verificar porta 8000 não está em uso
  → Verificar .env está preenchido
  → Rodar: pip install -r requirements.txt
