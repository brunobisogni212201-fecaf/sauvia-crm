🚀 QUICK START (8-10 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

1. Get Clerk Keys (2 min)
   → https://dashboard.clerk.com
   → Select "sauvia" organization
   → Settings → API Keys
   → Copy NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY and CLERK_SECRET_KEY

2. Run Setup Script (2 min)
   $ cd sauvia-app
   $ chmod +x scripts/clerk-mcp-setup.sh
   $ ./scripts/clerk-mcp-setup.sh
   → Follow prompts to enter Clerk keys

3. Start Services (3 terminals)
   Terminal 1: cd apps/backend && python -m uvicorn src.main:app --reload --port 8000
   Terminal 2: cd apps/web && pnpm dev
   Terminal 3: curl http://localhost:3000/.well-known/oauth-protected-resource/mcp

4. Test in Browser (2 min)
   → http://localhost:3000
   → Click Sign In
   → Complete Clerk auth
   → Verify Authorization header in DevTools

5. Verify MCP (1 min)
   → Call /mcp/http endpoint with Clerk token
   → Should return list of 9 tools
